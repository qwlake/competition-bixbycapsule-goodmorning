## Overview

이 캡슐은 간단한 사칙연산 캡슐입니다. 
이 캡슐을 통하여 Bixby에 대한 전반적인 기초 구조에 대하여 알 수 있습니다.

## Example flow

```
7 곱하기 7은?
16에서 7을 빼줘
```
